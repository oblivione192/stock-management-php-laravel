To set up the local environment first clone the git repository by running this command 
1) git clone https://github.com/oblivione192/stock-management-php-laravel.git 


2) After that run the command: 
cd stock-management-php-laravel

3) Run the command composer install. This will install all the packages for laravel backend.
4) Run the npm install. This will install all packages for the frontend.
5) Run php artisan migrate. This will create all the database tables for you.
6) To run the development server just run composer run dev

You are done! Have fun testing!
